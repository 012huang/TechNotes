

# gcc编译

## gcc

1. gcc 编译 C 程序

```
$ gcc tst.c -o tst
$ ./tst
```

2. gcc 编译 C++ 程序

```
$ gcc helloworld.cpp -lstdc++ -o helloworld
$ ./helloworld
```




## g++

1. g++ 编译 C++ 程序

```cpp
$ g++ helloworld.cpp -o helloworld
$ ./helloworld

$ g++ hellospeak.cpp speak.cpp -o hellospeak
$ ./hellospeak
```
